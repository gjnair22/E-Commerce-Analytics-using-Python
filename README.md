# E-Commerce Customer & Sales Analytics using Python

## Project Overview

This project analyzes the **Olist Brazilian E-Commerce Dataset** using Python to uncover customer behavior, sales performance, payment preferences, product trends, and delivery efficiency.

The project demonstrates a complete Data Analytics workflow including data loading, data cleaning, feature engineering, exploratory data analysis (EDA), business insight generation, and data visualization.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
